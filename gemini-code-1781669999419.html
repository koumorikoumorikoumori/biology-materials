【課題】特定の配列「GAATTC」を見つけて切断せよ！

制限酵素（ここではEcoRI）は、DNAの特定の塩基配列（回文配列）を認識して、決まった形で切断します。ボタンを押して、どのように切れるか観察しましょう。

5'- ... C G G A A T T C G C ... -3'
3'- ... G C C T T A A G C G ... -5'
断片A（左側）
5'- ... C G G
3'- ... G C C T T A A
粘着末端（切り口が噛み合う）
A A T T C G C ... -3'
         G C ... -5'

※切断ボタンを押すと、互い違いの切り口（粘着末端）ができることが分かります。ここに同じ酵素で切った別の遺伝子（インスリンなど）を近づけると、ぴったり結合（クローニング）できます。
